Privacy Policy

Last updated: July 10, 2026

The Breakup Reset tool is designed to be local-first. The private text you type into the Unsent Message and Reality Box tools stays in your browser unless you choose to export it.

1. What The Echo Box does

The Echo Box provides browser-based tools for moments when you need to pause before contacting someone. The current homepage includes an Unsent Message box, a ten-minute reset, a no-contact counter, and a Reality Box. The older Family Emergency Binder remains available at its own page.

2. Private text you type

For the Breakup Reset tool, your message and Reality Box notes are stored only in your browser's local storage if you allow saving. We do not operate a server database for this text, and this site does not send your message content to analytics, an AI provider, or a third-party checkout provider.

3. Local analytics

The Breakup Reset homepage currently stores basic product events locally in your browser, such as reset started, trigger selected, and counter saved. These local events do not include message body text, Reality Box content, names, phone numbers, or emails.

4. Payments

If you purchase the 30-Day No Contact Reset Kit, payment and file delivery are processed by Gumroad. We do not store your credit card details, Gumroad buyer records, order IDs, or payment information on this website.

5. Google Analytics and older pages

Some older Family Emergency Binder pages may use older analytics code. The Breakup Reset homepage currently uses a privacy-first local event interface while an external analytics provider is under review.

6. Sensitive information

Do not enter passwords, account secrets, government ID numbers, banking details, or other highly sensitive credentials into any The Echo Box tool. Browser local storage is convenient, but it is not a secure password manager.

7. Delete or export

The Breakup Reset tool includes buttons to delete or export local notes. You can also clear site data in your browser settings.
