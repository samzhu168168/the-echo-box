# Experiments

## Implemented in Batch A

A local A/B variant is persisted in localStorage:

- headline: pause or peace
- CTA: box or wait

The variant is stored only on the visitor's device and attached to local event records.

## Current limitation

There is no aggregate server collection yet, so this is a technical setup rather than a statistically useful experiment.
