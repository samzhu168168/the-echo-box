## ⚙️ 快速部署

### 1. 文件结构
```bash
your-project/
├─ index.html
├─ create.html
├─ style.css
├─ app.js
└─ assets/
   ├─ bg-cyber.png       # FutureBloom背景
   ├─ bg-vintage.png     # LoveScribe背景
   ├─ bg-gold.png        # Echo Box背景
   └─ framed-sample.png  # (可选) 首页展示图
```

### 2. 修改Gumroad链接
打开 `app.js` 第13-35行，替换为你的链接:
```javascript
gumroadLink: 'https://你的用户名.gumroad.com/l/你的产品ID'
```

### 3. 上传到Vercel
```bash
git add .
git commit -m "部署优化版"
git push
```

Vercel会自动检测并部署 (2分钟完成)
